2. Abstract
This system manages patient appointments, medicine inventory, and billing in a clinic or pharmacy. Using HTML for the front-end and MS Access for the database, it allows efficient record-keeping, appointment scheduling, stock tracking, and bill generation.

3. Introduction
Managing patient data, appointments, and medicines manually is time-consuming. This system automates these processes, improves accuracy, and ensures smooth clinic operations.

4. Objectives
•	Schedule and manage patient appointments.
•	Store and manage patient, doctor, and medicine information.
•	Track medicine stock levels and generate bills.
•	Simplify clinic operations using HTML and MS Access.

6. Database Tables (MS Access)
1. Patients: PatientID, Name, Age, Gender, Contact, Address
2. Doctors: DoctorID, Name, Specialization, Contact
3. Appointments: AppointmentID, PatientID, DoctorID, Date, Time
4. Medicines: MedicineID, Name, Category, StockQuantity, Price
5. Billing: BillID, PatientID, MedicineID, Quantity, TotalPrice, Date

7. HTML Website Pages
•	Homepage: Overview of clinic
•	Patients: Add/View patient records
•	Doctors: Add/View doctor records
•	Appointments: Schedule/View appointments
•	Medicines: Add/View stock
•	Billing: Create/View bills
•	About & Contact: Clinic info and contact form

8. Flowchart (Process Overview)
Patient Registration → Appointment Scheduling → Medicine Selection → Billing → Record Stored in Database

9. Conclusion
The system automates clinic management, improves efficiency, and ensures accurate record-keeping. It demonstrates practical use of HTML and MS Access in a professional setting.
